# Projeto de IA - Análise e Previsão de Gastos Pessoais

## Descrição
Este projeto tem como objetivo desenvolver uma solução baseada em Inteligência Artificial para analisar gastos pessoais e prever despesas futuras com base em um histórico de dados.
